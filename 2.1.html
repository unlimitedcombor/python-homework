题目 2.1: 判断点是否在第二或第四象限


#2.1
x, y = map(float, input().split(','))
print((x < 0 and y > 0) or (x > 0 and y < 0))

题目 2.2: 华氏度转摄氏度


#2.2
F = float(input())
C = 5/9 * (F - 32)
print(f"{C:.2f}")

题目 2.3: 三数之和与差


#2.3
a, b, c = map(int, input().split())
sum_val = a + b + c
sub_val = a - b - c
print(f"x1=a+b+c={a}+{b}+{c}={sum_val}")
print(f"x2=a-b-c={a}-{b}-{c}={sub_val}")

题目 2.4: 两数乘积、商和余数


#2.4
a, b = map(int, input().split())
product = a * b
quotient = a / b
remainder = a % b
print(f"{product} {quotient:.1f} {remainder}")

题目 2.5: 提取三位数的各位数字


#2.5
num = int(input())
a = num % 10
b = num // 10 % 10
c = num // 100
print(f"{a} {b} {c}")

题目 2.6: 复合赋值运算


#2.6
a = 2
a += a
a *= 2 + 3
a //= a + a
print(f"{a:.2f}")

题目 2.7: 格式化输出整数和浮点数


#2.7
# 读取输入
int_values = input().split()
float_values = input().split()

# 转换为相应的数据类型
a, b, c = map(int, int_values)
x, y, z = map(float, float_values)

# 格式化输出
# 整数部分：a=3   b=4   c=5
print(f"a={a:<7}b={b:<7}c={c:<7}")

# 浮点数部分：x=1.414  y=1.732  z=2.712
print(f"x={x:<7.3f}y={y:<7.3f}z={z:<7.3f}")

题目 2.8: 算术表达式求值


#2.8
result = 3.5 + (8/2 * (3.5 + 6.7) / 2) % 4
print(f"{result:.2f}")

题目 2.9: 计算圆的面积与周长


#2.9
r = float(input())
pi = 3.14159
area = pi * r * r
circumference = 2 * pi * r
print(f"{area:.2f} {circumference:.2f}")

题目 2.10: 整数转八进制和十六进制


#2.10
x = int(input())
print(oct(x), hex(x))
