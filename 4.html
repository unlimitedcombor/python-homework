题目 1: 计算自然常数e的近似值


#1
e = 1
term = 1
n = 1
error = 0.00001

while True:
    term *= 1 / n
    if term < error:
        break
    e += term
    n += 1

print(e)

题目 2: 找出1000以内的所有完数


#2
for i in range(2, 1000):
    sum_factors = 0
    for j in range(1, i // 2 + 1):
        if i % j == 0:
            sum_factors += j
    if sum_factors == i:
        print(i)

题目 3: 打印星号金字塔


#3
n = int(input())
i = 1
while i <= n:
    print('*' * i)
    i += 1
